The Questioners

Is a simple WEB application used for storing/creating/editing and deleting tasks and many more.

Built on a RESTful API server using Express, MongoDB and AngularJS.

It contains two main sections: To-do list and QaA(Questions and Answers).

Actively in development is a new section called Blog.

About

To-do list

Contains adding, checking, removing and permanently deleting tasks. Each task has a title, description and type with it's own defined color. Currently there are available only 3 types: work, personal and school . Checking means completing tasks, which visualy adds a line through text of the task. It is also possible to edit tasks. Tasks have 3 states: inprogress, completed and removed. It is possible to filter tasks by their type or state. Furthermore tasks have dates of their creation and completion. Deleting task permanently means deleting it off the database to never be retrieved again.
